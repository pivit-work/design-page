# 조직도 메뉴 UI 및 데이터 정책

## 개요
조직도(Org Chart) 메뉴 내 페이지들의 UI 규칙과 데이터 처리 정책을 정의한다.

---

## 권한별 표시 정책

### 일반 직원 (Employee)
- 멤버 카드에 **출퇴근 시간(workHours)** 까지만 표시
  - 근태 바 (attendance %)
  - 출퇴근 시간 (예: 9 → 6)
- 헬스 체크(HC Score)는 **표시하지 않음**

### 어드민 (Admin)
- 일반 직원에게 보이는 모든 정보 + **헬스 체크(HC Score)** 추가 표시
  - HC 점수 바 (0~10)
  - 점수 컬러: 7 이상 green / 5 이상 yellow / 5 미만 red

### 현재 구현 상태
- `adminMode` 토글로 헬스 체크 표시/숨김 처리 중 (`src/components/orgchart/OrgChartCanvas.jsx`)
- `showWorkHours` 토글로 출퇴근 시간 표시/숨김 처리 중 (`src/components/orgchart/OrgChartCanvas.jsx`)
- **현재는 토글로 모두 노출 가능** → 추후 로그인 권한 기반으로 전환 필요

### 향후 구현 방향
- 로그인 사용자의 role (`employee` / `admin`)에 따라 자동 분기
- 어드민 토글 UI는 어드민 로그인 시에만 노출
- 일반 직원 로그인 시 HC 관련 데이터 자체를 API 응답에서 제외 (보안)

## UI 정책

### 공통 규칙
- 컬러는 디자인 토큰(`var(--colors-xxx)`) 사용, 하드코딩 금지
- 멤버 카드 클릭 시 상세 모달 표시

### 직원 카드 표시 요소
- 아바타 우측 하단에 **상태 Dot** 표시 — 해당 직원의 재직 상태를 컬러로 나타냄
- 직원 이름 우측에 **직책 뱃지** 표시 — 본부장(DL) / 팀장(TL) / 파트장(PL)

### 사원 상태 Hover 정책
- 사원 카드에 마우스 Hover 시 해당 사원의 **재직 상태**가 표시된다
- 재직 상태는 4가지: **재직중 / 휴직 / 퇴사 예정 / 대기중**
- Hover 시 카드 보더가 상태별 컬러로 변경되고, 카드 하단 중앙에 상태 뱃지 표시

### 직원 프로필 모달 정책
- 직원 카드 클릭 시 프로필 모달이 표시된다
- 모달 내 직원의 **근무 상태** 뱃지가 표시되며, 3가지 상태가 있다: **근무중 / 회의중 / 집중모드**

#### 모달 권한별 표시 정보

**공통 (일반 직원 + 어드민)**
- 이름, 직책·소속, 자기소개, 근무 상태 뱃지
- 업무시간 (가운데 단독 표시)
- 피드백주기 / 미팅잡기 버튼

**어드민 전용 — 아이콘 영역**
- 업무시간 단독 → **고용형태 / 직급 / 업무시간** 3개로 확장 표시

**어드민 전용 — 추가 상세 정보**
- 사번
- 입사일
- 전화번호

#### 구현 상태
- `ProfileModal`에 `adminMode` prop으로 분기 구현 완료
- 일반: 업무시간 stat 1개만 표시
- 어드민: 고용형태/직급/업무시간 stat 3개 + 사번/입사일/전화번호 섹션 추가 표시
- 링크 섹션에 외부 링크 아이콘(arrow-up-right) 적용

### 텍스트 말줄임 규칙
- 캔버스 위 카드(회사 본부 카드, 직원 카드) 내 텍스트는 **최대 20자**
- 대상 필드: 회사명, 본부명, 직원명
- 20자 초과 시 `...` (ellipsis) 처리
- CSS: `text-overflow: ellipsis; overflow: hidden; white-space: nowrap;`

### 레이아웃
- 트리 구조 조직도 (BezierConnectors로 연결선 렌더링)
- 확대/축소, 드래그 이동 지원

### 컴포넌트
- `MemberCard` — 멤버 카드 (권한별 필드 분기)
- `BezierConnectors` — 조직도 연결선
- 프로필 모달 — OKR, HC, 1on1, 업무시간 통계 아이콘
