# 프로젝트 탭 UI 및 데이터 정책

## 개요
조직도 메뉴 내 프로젝트 서브탭의 UI 규칙과 데이터 처리 정책을 정의한다.

## 멤버 리스트 테이블 정책

### 초기 로드 및 더보기
- 처음 로드 시 **10명** 표시
- 더보기 버튼 클릭 시 **20명**씩 추가 표시
- 조직도 사원들이 상단에 먼저 표시되고, 이후 추가 데이터가 이어짐
